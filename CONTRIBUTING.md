# Contributing

Thank you for helping improve RaeburnAI Workflow Auditor.

## Development

```bash
npm install
npm run dev
npm run typecheck
npm run test
npm run build
```

## Good first contributions

- Add parsers for more document formats.
- Improve the opportunity taxonomy.
- Add export to PDF/Markdown.
- Add multi-tenant persistence with Postgres.
- Add benchmark datasets for audit quality.
- Improve accessibility and keyboard support.

## Pull request standards

- Keep changes focused.
- Add or update tests for business logic.
- Do not commit secrets or private process documents.
- Document new environment variables.
- Maintain TypeScript strict mode.

## Product principles

- Practical before flashy.
- Human-in-the-loop by default.
- Clear estimates with assumptions.
- Consultant-ready outputs.
- Open-source excellence.
